A sophisticated, modern web-based dashboard designed for managing bulk Gmail account creation, proxy monitoring, and automated email operations. This tool provides a centralized interface to oversee "farmed" accounts with integrated security and verification tracking.
## Core Features
 * Comprehensive Dashboard: Real-time visualization of account health, total account counts, and verification percentages using Chart.js.
 * Bulk Creation Suite: Tools for automated account generation (Premium feature).
 * Proxy Manager: Monitor proxy speeds and connectivity status to ensure account longevity and avoid IP flagging.
 * Account Manager: Organize and filter active versus flagged accounts with a clean, searchable interface.

## Visual Overview
### Dashboard Statistics
The interface tracks four primary metrics to ensure "farm" health:
 * Total Accounts: The aggregate number of profiles in your database.
 * Active Accounts: Accounts currently passing login checks.
 * Emails Sent: Global counter for outbound campaign volume.
 * Verified Percentage: Ratio of accounts with 2FA or SMS verification completed.
